## Installation

### Using CocoaPods

HDNTextField is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "HDNTextField"
```

### Using Carthage

Add the following to your Cartfile:

```bash
github "hdoria/HDNTextField"
```
